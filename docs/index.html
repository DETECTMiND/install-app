You'll see two security prompts. Both are normal for apps installed outside the Play Store.

Your browser will say "this type of file can harm your device" — tap Download anyway. Android will then say "this app was not checked for safety" — tap Install anyway.

Install

  1. Download using the button above.
  2. Open the file from your notification or your Files app → Downloads. If Android asks, allow your browser to install unknown apps, then come back.
  3. Install, then open the app. It will guide you through the consent screen, the permissions, and asking for your 6-character participant code.

Once you're set up, the app runs quietly in the background. You don't need to open it again unless asked.

What's collected

Recorded

  • App usage and screen state
  • Notifications metadata
  • Call and SMS frequency (hashed)
  • Location and ambient light
  • Taps and scrolls (no content)
  • Battery state
  • Ecological Momentary Assessment

Never recorded

  • Message or notification content
  • What you type or see on screen
  • Phone numbers in plain text
  • Photos, audio, or video
  • Web browsing history
  • Banking or payment activity
