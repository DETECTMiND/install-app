DETECTMiND
University of Southampton

A research app on smartphone behaviour and wellbeing.

Available to invited participants only. If you've been asked to take part, you can install the app below.

Download DETECTMiND.apk v1.0.0 · Android 8.0+ · ~20 MB
Heads up: Android will show security warnings during the first install — this is normal for apps distributed outside the Play Store. The steps below will get you through them.

Install

  1. Tap Download DETECTMiND.apk above. If the browser warns about the file type, choose Keep or Download anyway.
  2. Open the file from your notification shade, or from Files → Downloads.
  3. If Android asks, allow your browser to install unknown apps, then return to the installer.
  4. Tap Install, then Open.
  5. Read the consent screen carefully. If you're happy to take part, accept it.
  6. Grant the permissions the app asks for. Each one has a one-tap shortcut to the right Settings screen.
  7. Enter your 6-character participant code (sent to you separately) and you're done.

What's collected

The app records sensor data (such as app usage, screen state, gestures, and location) and your responses to short scheduled questions. Phone numbers and message content are SHA-256 hashed before storage — raw content is never saved.
